Infinity

The infinity brief was to design accessories for the Yourway infinity product range by Haworth. This project was a collaboration with my classmate Rajiv. I was project leader overseeing the project. We decided to design a mobile application for Haworth infinity range for creating bookings of single and group desks, as well as conference rooms. This application would also incorporate the ability to locate employees in a medium to large firm in an open plan / hot desking environment. The concept has been designed to be user friendly and easy to navigate.
