URN-IE - A NEW ALTERNATIVE

DESIGN BRIEF

Multiplier: The tension between singular and multiple what makes a singular object more value and how can value be added to the mass produced object.

DESIGN INTENT

To add value to a mass produced object by giving this object a very important and meaningful purpose, even though it is a mass produced item it will be an object with meaning.

The object will become a vessel to go through multiple processes, starting off as an indoor object then progressing outside where it will be reunited with other vessels that came from that production line to serve its most important purpose.
